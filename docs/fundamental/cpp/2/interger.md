# 整數

在電腦的世界中，我們使用0和1來儲存資料，我們把此稱為1個**bit**，有了一連串的**bit**，我們就可以二進位來儲存一個數字。

而電腦中的基本單位為 **8bits** ，我們稱其為1個**byte**。

了解到了這些基本知識，讓我們看看整數是如何存的。
